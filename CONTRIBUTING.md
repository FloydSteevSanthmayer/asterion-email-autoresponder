# Contributing to Asterion Email Autoresponder

Thanks for your interest in contributing! Please follow these guidelines:

1. Fork the repository and create a feature branch.
2. Write tests for any new behavior; run `pytest` before submitting a PR.
3. Keep code style consistent (we use `black` and `isort` via pre-commit).
4. Open a pull request with a clear description of your change.
5. For security fixes, please contact the maintainers privately and do not open a public PR.

We appreciate clear commit messages and small, focused PRs.
